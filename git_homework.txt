### **Задание:**
1. Откройте проект в **PyCharm** и запустите **Terminal** (находится внизу окна).
2. Проверьте текущий статус репозитория:

git status


---

### **Шаг 1: Создание папки и файла**
- Создайте новую папку **docs** в вашем проекте.
- Внутри папки **docs** создайте файл **readme.txt** и напишите краткую информацию о проекте.
- Добавьте и закоммитьте изменения:

git add docs/readme.txt
git commit -m "Добавлена папка docs и файл readme.txt с краткой информацией"


---

### **Шаг 2: Добавление нового файла**
- В папке **docs** создайте ещё один файл **about_me.txt** и напишите в нём несколько интересных фактов о себе (например, увлечения, любимые книги или фильмы).
- Добавьте и закоммитьте изменения:

git add docs/about_me.txt
git commit -m "Добавлен файл about_me.txt с интересными фактами о себе"


---

### **Шаг 3: Обновление файла**
- Откройте файл **readme.txt** и добавьте больше информации о проекте или измените текущий текст.
- Проверьте статус репозитория:

git status

- Добавьте и закоммитьте изменения:

git add docs/readme.txt
git commit -m "Обновлена информация в файле readme.txt"


---

### **Шаг 4: Отправка изменений на GitHub**

git push origin main  

Проверьте на GitHub, что папка **docs** и файлы внутри неё успешно отправлены.

---

**Формат сдачи:**
- Ссылка на репозиторий GitHub с обновлёнными файлами.
- После завершения задания отправьте ссылку на ваш репозиторий в **Telegram-группу**.
- Поиграйтесь с командами **git** и убедитесь, что вы понимаете, как они работают.
- Можете создать новые файлы, изменить существующие и отправить изменения на GitHub.
